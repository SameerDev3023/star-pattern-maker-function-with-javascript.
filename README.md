# star-pattern-maker-function-with-javascript.
this repo create a all  star patterns with javascript function created by me

In This Code User Can Easily Make Any Type of Advance Star Pattern Just Call function Which is created by me.

Function Name Is PatternMaker();


               ****************************** INFORMATION ABOUT CODE AND FUNCTION ****************************

                
1 ) This Function Required a Four Parameters First one is a Pattern Name For Example Star (*)............................

2 ) Second parameter is required pattern shape name for example octagon-square , pyramid , rhombus etc...................

3) Third parameter is required for Value of Row. You can set easily value of rows as per your choice.....................

4) Fourth  parameter is required for Value of Column. You can set easily value of columns as per your choice.............


                 ****************************** LIST OF SECOND PARAMETERS VALUES ****************************

   
                      1) square/ default ,  2) right-pyramid  , 3) reverse-right-pyramid ,   4) left-pyramid

                      5) pyramid  ,  6) octagon-square  , 7) cross  , 8) plus ,  9) spider  , 10) arrow 

                      11) kite , 12) empty-square-rhombus , 13) empty-rhombus , 14) rhombus-inside-empty

                      15) rhombus , 16 ) empty-square , 17 ) cross-empty-square, 18) tic-tac-toe , 19) diamond

                     20) left-arrow , 21) right -arrow
